# Домашнє завдання №8: Галерея зображень

Вітаю! Цей репозиторій містить виконання домашньої роботи №8 з курсу JavaScript.
Основна мета проєкту — створення інтерактивної галереї з використанням прийому
делегування подій та бібліотеки модальних вікон.

## 🚀 Основні цілі проєкту

- **Динамічна розмітка:** Генерація елементів галереї на основі масиву даних.
- **Делегування подій:** Оптимізація прослуховування кліків через контейнер
  `ul.gallery`.
- **Модальні вікна:** Інтеграція та використання бібліотеки `basicLightbox`.
- **Робота з атрибутами:** Використання `data-source` для передачі посилань на
  повнорозмірні зображення.

---

## 🛠 Технологічний стек

- **HTML5** (семантична розмітка)
- **CSS3** (Flexbox, Grid, стилізація згідно з макетом)
- **JavaScript (ES6+)** (DOM-маніпуляції, обробка подій)
- [**basicLightbox**](https://www.npmjs.com/package/basiclightbox?activeTab=dependencies)
  (зовнішня JS бібліотека)

---

## 📋 Виконані кроки (Декомпозиція)

1.  **Розмітка:** Створено контейнер `ul` з класом `gallery`.
2.  **Генерація контенту:** Написано скрипт для створення HTML-карток на основі
    масиву об’єктів `images`.
3.  **Стилізація:** Додано стилі згідно з наданим макетом для коректного
    відображення сітки.
4.  **Логіка кліку:** Реалізовано делегування на контейнер. Додано заборону
    переходу за посиланням за замовчуванням (`event.preventDefault()`).
5.  **Модальне вікно:**
    - Підключено `basicLightbox` через CDN.
    - Реалізовано відкриття модалки з великим зображенням при натисканні на
      картку.
    - Налаштовано підміну `src` зображення в модальному вікні.
